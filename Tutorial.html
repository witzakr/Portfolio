This is a tutorial prepared for our client. It includes a written manual and a video for them on how to use and modify the website. It's divided into two parts, one about setting up the website, adding images, and linking work. The second shows a detailed description of how to add an information/image slider page to the gallery picture.
These are the steps:
1st part:
- Installing the visual studio
- Installing necessary extensions
- Modifying and adding images
2nd part:
- adding the new HTML file
- using a ready template and copying it
- Checking if it works with extensions
- Adding text
Creating a user-friendly manual for our client is an important part of building the website. Our team worked together to achieve the goal of making sure the client can easily understand and use all the website features, including adding new content. Personally, this was my first time working on a team to create a manual, and it presented some challenges. However, I am proud to say that we successfully completed the task and produced a detailed manual for our client.
