I had this assignment where I had to create a detailed prototype for an online store, and I chose to design a shoe store using Axure, a program specifically for making prototypes.
This task was a bit tough for me because I wasn't very familiar with Axure, so I had to learn as I went along. The most challenging part was figuring out how to make a functional shopping cart. To solve this, I used global variables and repeaters to pass data between different pages. Once I got that sorted, I just had to finish up the rest of the pages, and I'm happy to say it turned out to be a successful prototype.
Despite the challenges, I enjoyed working on this assignment. It was a fun way to learn more about Axure and gain valuable experience in creating prototypes.
