For my Front-End development assignment, I had to build a web application for managing a library. The primary goal was to allow users to add, edit, and delete books, and then display them. The catch in this assignment was the use of APIs. We were required to create an API to store a JSON list of books, which the application could then manipulate.
However, I encountered a challenge with the website used to create the API. Once created, there was no direct way to modify the APIs. To work around this, I implemented a load button in the application. This button fetched data from the API and stored it in the local storage. From there, I could freely edit the data locally, overcoming the limitations of the API platform.
Despite the complications, the workaround allowed me to successfully manage and manipulate the library data within the scope of the assignment. Overall, this assignment was fun to make, it was my first time using API and I am happy with its outcome.
