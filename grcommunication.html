Communication section we can divide into two parts. The communication within the group and communication with a client. For the group part, we mostly communicate through WhatsApp and Discord. WhatsApp in our group is mostly responsible for fast communication, such as planning meetings or discussing project topics. Discord is used mostly to send important links or screenshots, plus we sometimes use its voice chat if we are doing work from home. For file sharing, we use SharePoint, because it allows us to work on the same file at the same time. Lastly, we also use Trello, which we use to divide tasks among the group members and to keep track of our progress. For the client communication, we communicate with them through email, where we usually send questions about their preferences regarding the website.
